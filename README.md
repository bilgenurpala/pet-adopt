# Pet Store 2026

VBT Internship 2026 project — a modern take on Swagger Petstore: FastAPI backend, Claude-powered AI assistant, web and mobile clients.

## Structure (monorepo)

- `backend/` — Pet Store API (FastAPI) — Bilge (service layer) + Seda (data layer)
- `ai/` — AI assistant service (FastAPI + Claude) — Bilge
- `frontend/` — Web app (Seda + Arjin)
- `mobile/` — Mobile app (Arjin)
